Sequências e Séries

O que são Sequências e Séries?

Uma sequência Imagine uma lista ordenada de números que segue uma regra específica. Cada número na lista é chamado de termo.

Ideia Central: É uma coleção ordenada de elementos (números, neste caso).
Analogia: É como uma fila de pessoas. A 1ª pessoa, a 2ª, a 3ª, e assim por diante. A ordem importa!
Notação: Geralmente representada por aₙ, onde n é a posição do termo (um número natural: 1, 2, 3...).

a₁ é o primeiro termo.
a₂ é o segundo termo.
aₙ é o n-ésimo termo (ou termo geral).

Uma série Agora, se pegarmos os termos de uma sequência e somarmos eles, nós temos uma série.

Ideia Central: É a soma dos termos de uma sequência.
Analogia: Usando a fila de pessoas, a série não é a lista de pessoas, mas sim o processo de contar quantas pessoas tem no total à medida que cada uma chega (1ª pessoa = 1, 1ª+2ª = 2, 1ª+2ª+3ª = 3, etc.).

Notação: A soma dos primeiros n termos de uma sequência aₙ é chamada de soma parcial e é denotada por Sₙ.

S₁ = a₁ (soma do primeiro termo)
S₂ = a₁ + a₂ (soma dos dois primeiros termos)
Sₙ = a₁ + a₂ + a₃ + ... + aₙ (soma dos n primeiros termos)
Uma Série Infinita é a soma de todos os infinitos termos de uma sequência. É representada por: a₁ + a₂ + a₃ + ... = ∑ (n=1 até ∞) aₙ

Principais Fórmulas

  • PA - Termo geral: aₙ = a₁ + (n - 1)·r
  • PA - Soma dos n primeiros termos: Sₙ = (a₁ + aₙ)·n / 2
  • PG - Termo geral: aₙ = a₁ · q^(n - 1)
  • PG - Soma dos n primeiros termos: Sₙ = a₁ · (qⁿ - 1) / (q - 1), q ≠ 1
