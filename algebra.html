Álgebra

O que é Álgebra?

A Álgebra é a parte da Matemática que usa letras e símbolos para representar números e expressar relações. Ela permite resolver problemas gerais e criar fórmulas que servem para qualquer valor.

Principais Conceitos

  • Distributiva: a(b + c) = ab + ac
  • Quadrado da soma: (a + b)² = a² + 2ab + b²
  • Quadrado da diferença: (a - b)² = a² - 2ab + b²
  • Equação do 1º grau: ax + b = 0 → x = -b/a
  • Equação do 2º grau: ax² + bx + c = 0 → x = (-b ± √Δ) / 2a
  • Delta (Δ): Δ = b² - 4ac
